定位技巧

跳转

1.两个项目窗口的跳转

Ctrl + Alt + [ 上一个窗口项目跳转

Ctrl + Alt + [ 下一个窗口项目跳转

2.一个项目中多个文件的跳转

Ctrl+E 查找最近文件

Ctrl+Shift+E 查找最近修改文件

3.文件内编辑位置定位

Ctrl+Shift+Backspace 上一次编辑定位

4.利用书签跳转

F11设立书签
Ctrl+F11+序号设立带序号书签

5.收藏文件和位置

Alt+2 调出收藏文件夹

Alt+Shift+f 添加收藏
